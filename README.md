# Contact Management System in C

A simple, file-based console application developed in C to manage personal contacts. This project demonstrates basic CRUD (Create, Read, Update, Delete) operations and file handling in C.

## 🚀 Features
- **Add Contact:** Save name, phone number, and email address.
- **View All Contacts:** List all saved contacts from the database file.
- **Search Contact:** Quickly find details by searching for a specific name.
- **Delete Contact:** Remove entries you no longer need.
- **Data Persistence:** All data is saved to a `.txt` or `.dat` file, so your contacts remain even after closing the program.
